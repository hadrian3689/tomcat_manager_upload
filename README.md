# Project Title

Tomcat WAR File Upload

## Description

Uploading a War file to the Tomcat Service and getting a pseudo-shell

### Dependencies

* pip install bs4

### Executing program

* Using python3
```
python3 tomcatmgr.py -t http://tomcatsite.blah -u username -p password --version 9
```

## Help

Help prompt
```
python3 tomcatmgr.py -h
```
